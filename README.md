## How to initiate a Typescript project?
> 

Run the follow commands:

- To create a node package 
`npm init -y`

- Install typescript, ts-node
`npm i typescript ts-node`

- After, initialize typescript
`npx tsc --init`

- And run with
`ts-node <filname>`
