## How to initiate a Typescript project?
> 

Run the follow commands:

- 1. To create a node package 
`npm init -y`

- 2. Install typescript, ts-node
`npm i typescript ts-node`

- 3. After, initialize typescript
`npx tsc --init`

- 4. And run your code with
`ts-node <filname>`
